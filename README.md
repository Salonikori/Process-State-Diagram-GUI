# Process State Diagram GUI

🔗 Live Demo: https://salonikori.github.io/Process-State-Diagram-GUI/

## Overview

Process State Diagram GUI is an interactive web-based visualization tool that demonstrates the lifecycle of a process in an Operating System. The project helps students understand how a process transitions between different states such as **New**, **Ready**, **Running**, **Waiting/Blocked**, and **Terminated**. These states are fundamental concepts in process management and CPU scheduling.

The application provides a graphical interface that makes learning process state transitions easier compared to static diagrams found in textbooks.

---

## Process States

The application demonstrates the following process states:

### New

The process is being created.

### Ready

The process is waiting to be assigned to the CPU.

### Running

The process is currently executing on the CPU.

### Waiting / Blocked

The process is waiting for an event such as I/O completion.

### Terminated

The process has finished execution and is removed from the system.
